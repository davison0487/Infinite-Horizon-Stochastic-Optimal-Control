# Infinite-Horizon Stochastic Optimal Control
We are trying to balance an inverted pendulum, we will model this as an infinite-horizon discounted stochastic optimal control problem. After finding the best policy for discretized pendulum state, we then solve this problem by interpolating this policy into continuous time and space.
